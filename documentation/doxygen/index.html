devIocStats EPICS Support Module

Introduction

devIocStats is a cross platform IOC statistics module. On VxWorks it replaces the vxStats support module.

It is developed and maintained at SLAC:
Link to devIocStats at SLAC.

Installing devIocStats on a linux IOC

The only template needed is ioc.db. This is used like:


file ioc.db
{
pattern {IOCNAME}
        {BL16I-MO-IOC-01}
}

The associated EDM screen is ioc_stats_soft.edl.

Installing devIocStats on a VxWorks IOC

The only template needed is ioc_vxworks.db. This is used like:


file ioc_vxworks.db
{
pattern {IOCNAME}
        {BL16I-MO-IOC-01}
}

The associated EDM screen is ioc_stats_vxworks.edl.
