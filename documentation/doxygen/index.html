devIocStats EPICS Support Module

Introduction

devIocStats is a cross platform IOC statistics module. On VxWorks it replaces the vxStats support module.

It is developed and maintained at SLAC:
Link to devIocStats at SLAC.

Installing devIocStats on a linux IOC

Use devIocStats.dbd and the devIocStats library, either libdevIocStats.a for static builds or libdevIocStats.so for dynamic builds.


file iocAdminSoft.db
{
	pattern
	{
		IOC
	}
        {
		<dom>-<tech>-IOC-01
	}
}
file iocAdminScanMon.db
{
	pattern
	{
		IOC
	}
        {
		<dom>-<tech>-IOC-01
	}
}

The associated EDM screen is ioc_stats_softdls.edl.

Installing devIocStats on a VxWorks IOC

Use devIocStats.dbd and the devIocStats library.


file iocAdminVxWorks.db
{
	pattern
	{
		IOC
	}
        {
		<dom>-<tech>-IOC-01
	}
}
file iocAdminScanMon.db
{
	pattern
	{
		IOC
	}
        {
		<dom>-<tech>-IOC-01
	}
}

The template iocGui.db contains beamlines information and includes deafults for all macros. The templates ioc.db and ioc_vxworks.db with the macro IOCNAME will be removed in the future. They are made by concatenating iocGui.db, iocAdminSoft.db or iocAdminVxWorks.db and iocAdminScanMon.db.
