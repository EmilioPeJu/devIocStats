00051 <p class="startli">Add the dependencies to configure/RELEASE. </p>

00064 <p class="startli">Add the LIBS dependencies to src/Makefile </p>

00067 </pre></div><p class="endli"></p>
00068 </li>
00069 <li>
00070 <p class="startli">Use the template files to add records to the database. </p>

00074 #  IOCNAME  Device prefix

00081 </pre></div><p class="endli"></p>
